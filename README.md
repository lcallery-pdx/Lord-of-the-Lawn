<h2>Description</h2>

"Lord of the Lawn" is a clicker-style game where you run a lawn mowing business. Each click mows one square foot of grass on a randomly generated lawn. As you progress, earn money and unlock upgrades like automated workers, faster equipment, and the ability to manage multiple lawns at once.
